# WeBlog

这是一个清新优美的博客系统。

## 项目用到的技术

- 语言：Python
- IDE：Pycharm
- Web框架：Flask
- 模板引擎：Jinja2
- 数据库连接：SQLAlchemy
- 数据库：MySQL
- 注册登录：Flask-Login
- Web应用服务器：Gunicorn
- HTTP服务器 ：Nginx
- 部分技术细节：
  - MVC，前后端分离
  - Flash Message传递消息
  - Flask-Script在命令行做一些管理操作
  - SQLAlchemy采用了对象关系映射（Object-Relational Mapping, ORM）
  - 密码salt加密
